# 05. 當日確認事項

### **【租⽤即視為明⽩並願意遵守空間使用手冊】**

### <mark style="color:blue;">務必攜帶物品</mark> <a href="#h.p_pseoqeq2vrz0" id="h.p_pseoqeq2vrz0"></a>

* 將投影機連接到 自己所攜帶之設備 時，如果您使用 HDMI 以外的設備進行連接，需自備轉接器。
* 活動指引告示物

### <mark style="color:blue;">緊急聯絡事宜</mark> <a href="#h.p_yf4mhmduplxk" id="h.p_yf4mhmduplxk"></a>

活動當日若有緊急設備問題發生，請與工作人員接洽或致電 04-22376859 與我們聯繫。

### <mark style="color:blue;">活動開始</mark> <a href="#h.p_axvhb5qcqmbr" id="h.p_axvhb5qcqmbr"></a>

預約時間到時，可以直接進入空間使用。如非於預約時間內到達會議室，可於抵達時致電 04-22376859，會由工作人員接待。

### <mark style="color:blue;">Wi-Fi無線網路</mark> <a href="#h.36jvhb9023rj" id="h.36jvhb9023rj"></a>

請告知所有活動參與者使用。\
SSID：WensCo文心會議室\
密碼：22376859

### <mark style="color:blue;">清潔</mark> <a href="#h.p_k85_x0z2xjt7" id="h.p_k85_x0z2xjt7"></a>

* 場地使用過程中，如有飲料食物弄髒地板，請立即通知工作人員，會提供清潔用具請使用者清潔乾淨。

### <mark style="color:blue;">遺失物</mark> <a href="#h.p_tkvsdbqg-oic" id="h.p_tkvsdbqg-oic"></a>

* 如果現場有任何遺留物品，務必請活動主辦方前來領取。
* 遺失物會暫時寄放於我司辦公室，如逾1個月無人領取，我們將不保留遺失物。

### <mark style="color:blue;">活動結束</mark> <a href="#h.p_qaco6p6vnkcj" id="h.p_qaco6p6vnkcj"></a>

當活動結束後，工作人員會確認設備是否完整無損壞，如有超時部分，也將依場地租借時段之金額加收費用。

### <mark style="color:blue;">協助配送行李</mark> <a href="#h.p_0l9k1c9q5yuj" id="h.p_0l9k1c9q5yuj"></a>

我司提供付費行李配送服務，可聯繫工作人員。

如您有配合的貨運公司，我司可提供短暫存放等待貨運公司領取。

### <mark style="color:blue;">如何處理垃圾</mark> <a href="#h.p_ofar84sytfys" id="h.p_ofar84sytfys"></a>

* 如何分類：請依照標示分類可回收垃圾（瓶、罐、PET瓶）和一般垃圾（其他）。
* 大件垃圾：請自行處理帶走。
* 如果您可能有大量垃圾：請務必與我們聯繫。
